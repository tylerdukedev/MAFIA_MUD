Capital Vice is installed.

Saves, layout, and archived playthroughs are stored here:
  %LOCALAPPDATA%\CapitalVice\

That folder is created the first time you run the game.

Tips:
  - Use Help > Manual in the game for controls and systems.
  - View > Reset Panel Layout restores the default dock layout.
  - F12 opens the developer console in preview builds.

Uninstall from Windows Settings or the Start menu entry.
